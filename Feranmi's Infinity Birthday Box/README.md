# Feranmi's Infinity Birthday Box

A deployable React/Vite birthday website using the photos in this folder.

## Run Locally

```bash
npm install
npm run dev
```

## Build For Deployment

```bash
npm run build
```

Deploy the generated `dist` folder to Netlify, Vercel, GitHub Pages, Cloudflare Pages, or any static host.

## Notes

- Optimized website images live in `public/media`.
- The visible copy is sourced from `Birthday Infinity Pic Box_V2.docx` and can be edited in `src/App.jsx`.
- No video files were found in the source folder at build time, so the experience currently uses still images.
